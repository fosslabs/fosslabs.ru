Ubuntu 11.10 INSTALL FEST Пн, 11 июня 2012, 17:00-18:00

610 аудитория, 2 корпус КФУ, Казань

Совсем недавно вышла новая версия Linux Ubuntu - Oneiric Ocelot 11.10. FOSS Labs проведет Install Fest, посвященный этому событию.

В программе:

  • "Что умеет Сновещательный Оцелот?"
  • "Что такое Ubuntu?"
  • "Семь лет истории Ubuntu"
  • "Kazan Ubunut LoCo Team"
  • Раздача дистрибутива, помощь в установке

Регистрация (бесплатная) обязательна (список участников передадут охране здания, чтобы всех пустили.
